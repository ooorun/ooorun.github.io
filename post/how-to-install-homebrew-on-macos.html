什么是 Homebrew

Homebrew 是一款自由及开放源代码的软件包管理系统，用以简化 macOS 系统上的软件安装过程。它拥有安装、卸载、更新、查看、搜索等很多实用的功能，通过简单的一条指令，就可以实现包管理，十分方便快捷。

安装 Homebrew

打开终端程序，在 Mac 上打开或退出“终端”：

  • 启动台 > 其他 > 终端
  • 启动台 或是　Commond⌘ + 空格 打开聚焦搜索，搜索 终端 或是 Terminal
